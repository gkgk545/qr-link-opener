QR Link Opener

개인정보처리방침 (Privacy Policy)

최종 수정일: 2025년 1월 4일

1. 개요

QR Link Opener(이하 "본 확장 프로그램")는 사용자의 개인정보를 소중히 여기며, 개인정보 보호에 최선을 다하고 있습니다.

2. 수집하는 정보

본 확장 프로그램은 어떠한 개인정보도 수집하지 않습니다.

  • 사용자 데이터를 수집하지 않습니다
  • 브라우징 기록을 저장하지 않습니다
  • 외부 서버로 데이터를 전송하지 않습니다

3. 권한 사용 목적

본 확장 프로그램은 다음 권한을 오직 QR 코드 스캔 기능을 위해 사용합니다:

  • activeTab: 사용자가 아이콘 클릭 시 현재 탭에서 QR 코드 스캔
  • scripting: QR 코드 선택 UI 표시
  • host_permissions: 모든 웹사이트에서 QR 코드 스캔 지원

4. 데이터 저장

본 확장 프로그램은 어떠한 데이터도 로컬 또는 원격 서버에 저장하지 않습니다. QR 코드 스캔 결과는 즉시 처리되며 저장되지 않습니다.

5. 제3자 공유

본 확장 프로그램은 어떠한 정보도 제3자와 공유하지 않습니다.
